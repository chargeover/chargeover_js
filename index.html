ChargeOver.js Examples

This GitHub repository shows examples of using the ChargeOver.js library to securely take signups and allow customers to enter credit card details in a PCI compliant manner.

Getting Started

Please note that for each example, you'll have to swap out the configuration for your own configuration. You can find the configuration code within your ChargeOver account, by going to:

  • Configuration > API & Web Hooks

Edit the example file and swap out the Javascript before trying these examples.

Helpful Links

  • This GitHub repository: https://github.com/chargeover/chargeover_js_example
  • ChargeOver.js documentation: https://developer.chargeover.com/apidocs/js/
  • For support: support@ChargeOver.com

File Topic Description
signup_w_credit_card.html Basics This example illustrates a basic signup form, securely collecting credit card details in a PCI compliant manner.
signup_without_payment.html Basics This example illustrates a basic signup form, that DOES NOT require any payment information. i.e. it creates a subscription in ChargeOver, but doesn't collect any payment information up front.
signup_w_initial_promo_discount.html Discounts This example illustrates a signup form where the first few months of service are given at a discounted rate.
signup_w_addons.html Add-Ons This example illustrates a signup form where customers are encouraged/upsold by adding optional add-ons to their subscription.
signup_w_multiple_plans.html Advanced This example illustrates a signup form where the customers signs up for multiple plans/products at once (e.g. a bundle or group of services).
